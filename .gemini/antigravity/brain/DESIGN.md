# UI/UX Design

## Design Philosophy

- Minimalist
- Zero distraction
- Fast interaction
- Low cognitive load
- Developer-first aesthetic
- System-native behavior

---

## UI Components

- Floating always-on-top popup
- Large circular mic button
- Language dropdown
- Transcription output area
- “Convert to Prompt” CTA button
- Status indicator dot

---

## System-Level Behavior

- Runs silently in Windows background
- Activated via global shortcut
- Popup appears instantly
- After speech completion:
    - Text copied to clipboard
    - Simulated paste into active application
    - Popup auto-minimizes
