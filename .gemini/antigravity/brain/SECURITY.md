Act as a senior security engineer and ethical AI auditor.

# Security Architecture

## Threat Model

- API key leakage
- Audio data interception
- Prompt injection attacks
- Clipboard misuse
- Malicious shortcut triggering

---

## Security Controls

- Encrypted API storage
- Secure transport (HTTPS)
- Input sanitization
- Permission-based microphone access
- Rate limiting
- Background process monitoring

---

## Privacy Policy

- No audio stored permanently
- No speech stored without consent
- No prompt history stored without user approval
- Local processing preferred where possible
