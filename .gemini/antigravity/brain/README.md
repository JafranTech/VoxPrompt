# VoxPrompt AI

VoxPrompt AI is a multilingual voice-to-prompt intelligent Windows desktop agent designed to convert speech into structured, LLM-optimized prompts and inject them system-wide.

---

## Overview

VoxPrompt runs as a background Windows desktop application and can inject text into any currently focused input field across applications (browser, IDE, chat apps, etc.).

The system enables users to:

- Activate voice input via global shortcut
- Record continuous speech (no auto-stop)
- Convert English speech → English text
- Convert Tamil speech → Romanized Tamil (Tanglish)
- Transform raw speech into structured AI-ready prompts
- Automatically paste generated output into the active application

VoxPrompt bridges the gap between natural human speech and structured AI communication.

---

## Core Value Proposition

Unlike traditional dictation tools, VoxPrompt:

- Works system-wide (not limited to browser)
- Runs in background like native Windows utilities
- Injects text into any focused input field
- Adds an AI Prompt Optimization Layer
- Provides Tamil → Roman transliteration
- Supports voice-command-based transformations

VoxPrompt AI is not just voice typing.
It is a Voice-Driven System-Level Prompt Engineering Agent.
